
# 영양밥
* 완성된 영양밥에 부추양념장, 참기름, 깨소금, 김가루를 뿌려 먹으면 더 맛있다.

## 표고버섯 영양밥
* 표고버섯, 느타리 버섯, 당근, 연근으로 밥을 한다.

## 전복밥
* 전복으로 밥을 한다.
* 전복밥은 전복죽의 재료준비 과정과 비슷하며, 밥솥에 물을 더 적게 넣고한다는 것이 다르다.
